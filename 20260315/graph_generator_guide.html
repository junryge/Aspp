01

설치 & 의존성

graph_generator.py는 단일 파일 모듈입니다. 프로젝트 디렉토리에 복사해서 바로 사용하세요.

필수 패키지

pip install matplotlib numpy
한글 폰트 자동 감지: 시스템에 NanumGothic, Malgun Gothic, Noto Sans KR 등이 설치되어 있으면 자동으로 한글이 깨지지 않습니다.
02

빠른 시작

가장 간단한 사용법 3줄:

from graph_generator import GraphGenerator

gen = GraphGenerator()
gen.bar(labels=["1월", "2월", "3월"],
        datasets=[{"label": "매출", "data": [120, 190, 300]}])
gen.save("output.png")

이 코드만으로 막대 그래프 PNG가 생성됩니다.

03

간단 API 방식

GraphGenerator 인스턴스에서 차트 메서드를 직접 호출합니다. 모든 메서드는 self를 반환하므로 체이닝이 가능합니다.

gen = GraphGenerator(figsize=(12, 7), dpi=150)

# 메서드 체이닝
gen.line(
    labels=["Q1", "Q2", "Q3", "Q4"],
    datasets=[
        {"label": "매출", "data": [100, 200, 180, 350], "borderColor": "#6366f1"},
        {"label": "비용", "data": [80, 120, 110, 200], "borderColor": "#ec4899"},
    ],
    title="분기별 실적",
    fill=True
).save("quarterly.png")

메서드 시그니처 공통 패턴

파라미터타입설명
labelslist[str]X축 레이블 (카테고리 이름)
datasetslist[dict]데이터셋 배열. 각 dict에 "data", "label", 색상 키 포함
titlestr | None차트 제목 (선택)
04

Chart.js Config 방식

Chart.js와 동일한 JSON 구조를 그대로 사용합니다. 웹 프론트엔드에서 쓰던 config를 Python 서버에서 이미지로 렌더링할 때 유용합니다.

gen = GraphGenerator()
gen.from_chartjs_config({
    "type": "bar",
    "data": {
        "labels": ["A", "B", "C"],
        "datasets": [{"label": "값", "data": [10, 20, 30]}]
    },
    "options": {
        "plugins": {"title": {"display": True, "text": "샘플 차트"}},
        "indexAxis": "y"  # 가로 막대
    }
})
gen.save("chart.png")

지원되는 options 키

옵션 경로효과
options.plugins.title.text차트 제목 표시
options.indexAxis: "y"가로 막대 그래프로 변환
options.scales.x.stacked: true누적 막대 그래프
JSON 문자열도 OK: from_json_string(json_str) 또는 from_chartjs_config(json_str)에 문자열을 넘기면 자동 파싱됩니다. 한 줄 주석(//)과 trailing comma도 자동 제거.
05

차트 유형별 사용법

▮ bar — 막대 그래프

gen.bar(
    labels=["1월", "2월", "3월"],
    datasets=[
        {"label": "매출", "data": [120, 190, 300], "backgroundColor": "#6366f1"},
        {"label": "비용", "data": [80, 100, 150], "backgroundColor": "#ec4899"},
    ],
    title="월별 비교",
    horizontal=False,  # True면 가로 막대
    stacked=False       # True면 누적
)

╱ line — 꺾은선 그래프

gen.line(
    labels=["Q1", "Q2", "Q3", "Q4"],
    datasets=[{"label": "성장률", "data": [2.5, 3.1, 4.2, 3.8]}],
    title="분기별 성장",
    fill=True,       # 영역 채우기
    markers=True    # 데이터 포인트 마커
)

◕ pie / doughnut — 원형 · 도넛

# 원형
gen.pie(
    labels=["Python", "JS", "Go"],
    datasets=[{"data": [45, 35, 20]}]
)

# 도넛 (가운데 비어있음)
gen.doughnut(
    labels=["완료", "진행", "대기"],
    datasets=[{"data": [60, 25, 15]}]
)

⬡ radar — 레이더 차트

gen.radar(
    labels=["속도", "안정성", "확장성", "보안", "비용"],
    datasets=[
        {"label": "제품A", "data": [90, 70, 85, 60, 40]},
        {"label": "제품B", "data": [60, 90, 70, 85, 80]},
    ]
)

⋯ scatter — 산점도

# dict 형식 (x, y 좌표)
gen.scatter(
    labels=[],
    datasets=[{
        "label": "샘플",
        "data": [{"x": 1, "y": 10}, {"x": 3, "y": 25}, {"x": 5, "y": 18}]
    }]
)

# 리스트 형식 (x는 자동 인덱스)
gen.scatter(labels=[], datasets=[{"data": [10, 25, 18, 30]}])

◉ bubble — 버블 차트

gen.bubble(
    labels=[],
    datasets=[{
        "label": "프로젝트",
        "data": [
            {"x": 10, "y": 20, "r": 5},   # r = 버블 크기
            {"x": 30, "y": 40, "r": 10},
            {"x": 50, "y": 15, "r": 8},
        ]
    }]
)

◔ polarArea — 극좌표 영역

gen.polar_area(
    labels=["R&D", "마케팅", "영업", "인사"],
    datasets=[{"data": [300, 200, 250, 150]}]
)
06

출력 메서드

💾

save(filepath)

PNG, SVG, PDF, JPG 등으로 파일 저장. 확장자로 포맷 자동 인식.

🔗

to_base64(fmt)

base64 문자열 반환. <img src="data:image/png;base64,..."> 형태로 웹 임베딩.

📦

to_bytes(fmt)

바이너리 bytes 반환. HTTP Response나 파일 스트리밍에 사용.

🖥️

show()

GUI 환경에서 matplotlib 창으로 미리보기.

# 파일 저장
gen.save("chart.png")
gen.save("chart.svg", transparent=True)

# base64 (웹 임베딩)
b64 = gen.to_base64()
html = f'<img src="data:image/png;base64,{b64}">'

# bytes (FastAPI 응답)
from fastapi.responses import Response
data = gen.to_bytes()
return Response(content=data, media_type="image/png")
주의: 차트 메서드를 먼저 호출하지 않고 save(), to_base64() 등을 호출하면 RuntimeError가 발생합니다.
07

편의 함수

클래스 인스턴스 없이 한 줄로 차트를 생성하는 모듈 레벨 함수들입니다.

from graph_generator import generate_chart, quick_bar, quick_line, quick_pie

# 1) generate_chart — Chart.js config 기반
generate_chart({
    "type": "bar",
    "data": {"labels": ["A","B"], "datasets": [{"data": [10,20]}]}
}, "out.png")  # output_path 없으면 base64 반환

# 2) quick_bar / quick_line / quick_pie — 최소 파라미터
quick_bar(["A", "B", "C"], [10, 20, 30], title="제목", output_path="bar.png")
quick_line(["1", "2", "3"], [5, 15, 10], output_path="line.png")
quick_pie(["A", "B"], [60, 40], output_path="pie.png")

# output_path 생략 → base64 문자열 반환
b64 = quick_bar(["X", "Y"], [100, 200])
함수파라미터반환값
generate_chart(config, output_path, figsize, dpi)Chart.js JSON dict 또는 str파일경로 또는 base64
quick_bar(labels, data, title, output_path)labels: list, data: list파일경로 또는 base64
quick_line(labels, data, title, output_path)labels: list, data: list파일경로 또는 base64
quick_pie(labels, data, title, output_path)labels: list, data: list파일경로 또는 base64
08

기본 색상 팔레트

색상을 지정하지 않으면 아래 10색이 순서대로 자동 적용됩니다.

#6366f1 인디고 #8b5cf6 보라 #ec4899 핑크 #f59e0b 앰버 #10b981 에메랄드 #3b82f6 블루 #ef4444 레드 #84cc16 라임 #14b8a6 틸 #f97316 오렌지

직접 색상을 지정하려면 dataset dict에 "backgroundColor" 또는 "borderColor" 키를 추가하세요:

{"label": "커스텀", "data": [10, 20], "backgroundColor": "#ff6b6b"}

# pie/doughnut은 리스트로 개별 색상 지정 가능
{"data": [30, 70], "backgroundColor": ["#6366f1", "#ec4899"]}
09

생성자 옵션

gen = GraphGenerator(
    figsize=(10, 6),   # (가로, 세로) 인치
    dpi=100,            # 해상도 (기본 100, 고화질은 150~200)
    style="default"      # matplotlib 스타일 (예: "dark_background", "ggplot")
)
파라미터기본값설명
figsize(10, 6)그래프 크기 (인치). 큰 차트는 (14, 8) 등
dpi100해상도. 프레젠테이션용은 150, 인쇄용은 300
style"default"matplotlib 내장 스타일. "dark_background", "seaborn-v0_8", "ggplot" 등
