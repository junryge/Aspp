3.2 핵심 알고리즘 수식

① CNN 특징 추출

합성곱 연산
yi = f(∑j wj × xi+j + b)

여기서 w는 필터 가중치, x는 입력 시퀀스, b는 편향, f는 활성화 함수(ReLU)

② LSTM 시계열 모델링

LSTM 게이트 연산
ft = σ(Wf · [ht-1, xt] + bf) (망각 게이트)
it = σ(Wi · [ht-1, xt] + bi) (입력 게이트)
Ct = ft × Ct-1 + it × tanh(WC · [ht-1, xt] + bC)
ot = σ(Wo · [ht-1, xt] + bo) (출력 게이트)
ht = ot × tanh(Ct)

③ Multi-Task 손실 함수

통합 손실 함수
Ltotal = α × Lregression + β × Lclassification

Lregression = (1/n) ∑(ypred - ytrue)² (MSE)
Lclassification = -∑ ytrue × log(ypred) (Cross-Entropy)

α = 0.7, β = 0.3 (가중치)

🔧 특징 엔지니어링

4.1 시간적 특징

  • 이동 평균 (MA): 5분, 10분, 30분, 60분 이동평균으로 단기/장기 트렌드 파악
  • 표준편차 (STD): 5분, 10분, 30분 표준편차로 변동성 측정
  • 변화율: 1분, 5분, 10분 단위 변화율 및 가속도 계산
  • 최대/최소값: 10분 윈도우 내 최대/최소값으로 극값 파악

4.2 공간적 특징

  • 경로별 불균형: 팹 간 양방향 물류량 차이 계산
  • 부하율: 각 팹의 입출력 물류 부하 비율
  • 경로별 비율: 전체 물류량 대비 각 경로의 비중
  • 병목 지표: 임계치 초과 여부 및 위험도 점수

4.3 특징 계산 예시

# 팹 간 불균형 지표 계산 imbalance_M14A_M10A = data['M14AM10A'] - data['M10AM14A'] imbalance_M14A_M14B = data['M14AM14B'] - data['M14BM14A'] imbalance_M14A_M16 = data['M14AM16'] - data['M16M14A'] # 부하율 계산 load_M14A_out = (data['M14AM10A'] + data['M14AM14B'] + data['M14AM16']) / total_count load_M14A_in = (data['M10AM14A'] + data['M14BM14A'] + data['M16M14A']) / total_count # 변화율 및 가속도 change_rate = data['TOTALCNT'].pct_change() acceleration = change_rate.diff()

7.1 모델 사용 방법

# 모델 로드 model = load_model('model/cnn_lstm_multitask_final.keras') scaler = joblib.load('scaler/multitask_scaler.pkl') # 실시간 데이터 전처리 current_data = get_realtime_data() # 최근 60분 데이터 features = create_features(current_data) scaled_features = scaler.transform(features) # 예측 수행 predictions = model.predict(scaled_features) logistics_volume = predictions[0][0] # 물류량 bottleneck_probs = predictions[1][0] # 병목 확률 # 결과 해석 bottleneck_location = ['없음', 'M14A-M10A', 'M14A-M14B', 'M14A-M16'][np.argmax(bottleneck_probs)] confidence = np.max(bottleneck_probs) print(f"10분 후 예상 물류량: {logistics_volume:.0f}개") print(f"병목 예상 위치: {bottleneck_location} (신뢰도: {confidence:.1%})")

# 1분 후 예측으로 변경 def create_targets(data, future_minutes=1): # 10에서 1로 변경 data['FUTURE_TOTALCNT'] = pd.NA for i in data.index: future_time = i + pd.Timedelta(minutes=future_minutes) if future_time in data.index: data.loc[i, 'FUTURE_TOTALCNT'] = data.loc[future_time, 'TOTALCNT'] return data
