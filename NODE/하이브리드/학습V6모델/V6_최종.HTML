3. 모델별 예측 방법 (쉬운 설명)

📌 Model 1: LSTM - 장기 기억 모델

예측 공식
f₁ = 0.4 × (현재값) + 0.3 × (과거평균) + 0.3 × (추세×10분)
예시:
• 현재 M14AM14B = 320
• 과거 100분 평균 = 250
• 상승 추세 = +2/분
• 예측 = 0.4×320 + 0.3×250 + 0.3×(2×10) = 1,509

📌 Model 2: GRU - 단기 변화 감지

예측 공식
f₂ = 현재값 + (최근변화율 × 10분)
예시:
• 현재 TOTALCNT = 1,420
• 10분 전 = 1,380
• 변화율 = (1,420-1,380)÷10 = 4/분
• 예측 = 1,420 + (4×10) = 1,460

📌 Model 3: CNN-LSTM - 패턴 매칭

예측 공식
f₃ = 유사패턴_평균 × 현재패턴_강도
예시:
• 과거 유사 패턴 평균 = 1,500
• 현재 패턴 일치도 = 95%
• 예측 = 1,500 × 0.95 = 1,425

📌 Model 4: Spike Detector - 급증 감지
