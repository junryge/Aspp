## 📌 요청 목적

LLM(대규모 언어 모델)이 Logpresso 쿼리를 자동 생성할 수 있도록 학습 데이터를 구축하고자 합니다.
이를 위해 아래 정보가 필요합니다.

**목표**: 사용자가 자연어로 질문하면 → LLM이 Logpresso 쿼리를 생성

```
예시)
사용자: "M14 FAB의 오늘 반송량 알려줘"
LLM 생성: table duration=1d ts_current_job | search FAB == "M14" | stats count
```
