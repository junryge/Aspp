[예시 1: M16A_LIFTER_Normal_LFT_Only.csv - 정상 이송]
--------------------------------------------------------------------------------
파일 특성:
  - 레코드: 22건
  - 메시지: STORAGE-ZIPTOWER* (리프터 전용)
  - 시간: 17:44:07 ~ 17:46:27 (약 2분 20초)
  - 캐리어: 4PDML395
  - 장비: 6ABL6022
  - 경로: 6ABL6022_AI311 (3층) → 6ABL6022_AOG6 (6층)

사용자 질문: "이 파일 분석해줘"

응답 예시:
```
이 파일은 M16A 리프터의 층간 이송 로그예요.

캐리어 4PDML395가 리프터를 타고 3층에서 6층으로 이동한 기록인데요:

📍 경로: AI311 (3층 입구) → AOG6 (6층 출구)
🏗️ 장비: 6ABL6022 (M16A 리프터)
⏱️ 소요시간: 약 2분 20초
✅ 결과: 정상 완료 (RESULTCODE=0)

이벤트 순서:
1. 17:44:07 - 3층 입구에서 바코드 스캔
2. 17:44:07 - 입고 대기 상태
3. 17:44:08 - 이송 명령 발행 (AI311 → AOG6)
4. 17:44:08 - 크레인 동작 시작
5. 17:45:12 - 이송 중 (층간 이동)
6. 17:46:00 - 6층 도착, 위치 변경
7. 17:46:04 - 출고 대기, 이송 완료
8. 17:46:27 - OHT로 캐리어 전달 (HANDOFFTYPE=2)

모든 이벤트가 WELL(정상)이고 시퀀스도 정상이에요.
3층→6층 층간 이송이 문제없이 완료된 케이스입니다.
```

[예시 2: LIFTER_with_FABJOB - FABJOB 포함]
--------------------------------------------------------------------------------
파일 특성:
  - 레코드: 88건
  - 메시지: STORAGE-* + VM-* 혼합
  - FAB 이동 중 리프터 구간

사용자 질문: "전체 진행 상황 알려줘"

응답 예시:
```
이 파일은 리프터 이벤트 + FAB간 이송 JOB 정보가 함께 있어요.

📦 캐리어: 4PDML395
📍 FAB 이동: M14A(3층) → M16A(6층) 중 리프터 구간

리프터 구간 상세:
- 입구: 6ABL6022_AI311 (17:44:07)
- 출구: 6ABL6022_AOG6 (17:46:04)
- 소요: 약 1분 57초

FAB JOB 상태:
- VM-FABTRANSPORTLOCATIONCHANGED: 위치 보고 중
- 리프터 구간 완료 후 M16A OHT로 전달 예정

✅ 리프터 구간 정상 완료
⏳ 이후 M16A OHT → Stocker 구간 진행
```

[예시 3: 크레인 지연]
--------------------------------------------------------------------------------
파일 특성:
  - CRANEACTIVE → CRANEIDLE 간격이 긴 경우

사용자 질문: "왜 오래 걸렸어?"

응답 예시:
```
⚠️ 리프터 이송이 평소보다 오래 걸렸어요.

📍 구간: AI311 → AOG6
⏱️ 소요시간: 5분 30초 (정상: 2분)

지연 구간 분석:
- CRANEACTIVE → TRANSFERRING: 3분 20초 (지연!)
- TRANSFERRING → COMPLETED: 정상

크레인 동작 시작 후 실제 이송까지 지연이 발생했어요.

가능한 원인:
1. 다른 캐리어가 리프터 사용 중
2. 크레인 대기열 정체
3. 층간 이동 경로에 다른 작업

💡 해당 시간대 리프터 사용량 확인이 필요해요.
```

[예시 4: 바코드 에러]
--------------------------------------------------------------------------------
파일 특성:
  - IDREADSTATUS가 0이 아닌 경우

사용자 질문: "에러 분석해줘"

응답 예시:
```
⚠️ 리프터 입구에서 바코드 스캔 에러가 발생했어요.

에러 내용:
- 위치: 6ABL6022_AI311
- IDREADSTATUS: 1 (읽기 실패)
- 시간: 17:44:07

이후 처리:
- 재스캔 시도 후 성공
- 이송은 정상 진행됨

🔍 확인 필요:
1. 캐리어 바코드 라벨 상태
2. 리프터 입구 스캐너 청소 상태
3. 캐리어 진입 각도/위치
```
