## 🎯 모델 성능 요약
| 지표 | 값 | 판정 |
|---|---|---|
- 정확도, 재현율, 정밀도, F1-Score

## 📊 혼동 행렬 분석
- TP(정탐): 위험 예측 → 실제 위험 ✅
- TN(정상): 정상 예측 → 실제 정상 ✅
- FP(오탐): 위험 예측 → 실제 정상 ⚠️
- FN(미탐): 정상 예측 → 실제 위험 🔴

## 📈 급증 컬럼과 예측 정확도 (SPIKE_INFO)
| 급증 패턴 | 예측 적중률 | FN 발생 | FP 발생 |
|---|---|---|---|
- 급증 컬럼 多 → 예측 적중률 높음/낮음?
- FN 발생 시 급증 컬럼 특징
- FP 발생 시 급증 컬럼 특징

## 🔮 상태 예상 정확도 (STATUS_PREDICTION)
| 상태 | 실제 1700+ 발생률 | 판정 |
|---|---|---|
- 병목예상 → 실제 위험 발생률
- 위험예상 → 실제 위험 발생률
- 상태 예상 vs 10분/30분 알람 일치도

## 🔔 알람 vs 예측 비교
- 10분/30분 예측 알람 정확도
- 로그프레소 알람과의 일치율

## ⚠️ 개선 필요 사항
- FN 감소 방안 (놓친 위험 줄이기)
- FP 감소 방안 (오탐 줄이기)
- 급증 컬럼 활용한 예측 정확도 향상 방안
