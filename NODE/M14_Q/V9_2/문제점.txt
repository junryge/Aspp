1. 기존에 뭘 잘못했나?
================================================================================

[문제 1] 쓸데없이 복잡했음

    기존                    문제
    --------------------------------------------------
    Feature 82개            대부분 쓸모없음
    280분 시계열            너무 김, 60분이면 충분
    1700+ 데이터 50배 증강  오히려 해로움


[문제 2] 황금패턴이 틀렸음

    기존 황금패턴: M14AM14B > 540 AND M14AM14BSUM > 620
    
    결과: 감지율 71.5%
    → 100개 중 71개만 잡음
    → 29개 놓침!


[문제 3] "예측"이 아니었음

    충격적인 사실:
    
    11월 8일 ~ 12월 11일 데이터 전체 분석함.
    
    현재 TOTALCNT가 1550 미만인데 10분 후 1700+ 된 경우?
    
    → 0건
    
    무슨 말이냐면:
    - 10분 만에 갑자기 1400 → 1700 이런 일은 없었음
    - 1700 되려면 이미 1550 이상이었음
    - 즉, "예측"이 아니라 "이미 높은 거 감지"였음


================================================================================
2. 그래서 뭘 바꿨나?
================================================================================

[변경 1] 타겟을 바꿈

    기존: 10분 후 TOTALCNT 값을 직접 예측
    변경: 변화량만 예측 (10분 후 - 현재)
    
    왜?
    → 현재값이 제일 중요함
    → 변화량만 알면 됨
    → 예측값 = 현재값 + 변화량


[변경 2] Feature 줄임

    기존: 82개
    변경: 33개
    
    왜?
    → 대부분 쓸모없었음
    → 현재값 + 최근 추세만 중요


[변경 3] 시계열 줄임

    기존: 280분
    변경: 60분
    
    왜?
    → 4시간 전 데이터가 10분 후 예측에 도움 안 됨
    → 최근 1시간이면 충분


[변경 4] 데이터 증강 제거

    기존: 1700+ 데이터 50배 복사
    변경: 안 함
    
    왜?
    → 과적합만 유발
    → 원본 데이터로 학습


[변경 5] 새 컬럼 추가

    추가: M16M14A (M16에서 M14A로 역류량)
    
    왜?
    → 12월 11일 분석에서 발견
    → 1700+ 케이스에서 항상 140 이상이었음


[변경 6] 위험 판단 기준 변경

    기존: M14AM14B > 540 AND M14AM14BSUM > 620
    변경: TOTALCNT >= 1580
    
    결과:
    - 기존: 감지율 71.5%
    - 변경: 감지율 100%
