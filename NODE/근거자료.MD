## 6. 5점 (지능화) 상세

### 5점이 뭐냐면

```
5점 = "사람 개입 없이 알아서 돌아가는 시스템"

┌─────────────────────────────────────────────────────┐
│                                                     │
│   예측 수행 → 자동 조치 → 결과 수집 → 자동 재학습   │
│       ↑                                      │      │
│       └──────────────────────────────────────┘      │
│                    피드백 루프                       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 5점 시나리오 예시

```
1. 모델이 "10분 후 급증 예측" 출력
      ↓
2. 자동 제어: OHT 속도 조절 / 경보 발송 (룰 기반)
      ↓
3. 10분 후 실제 결과 자동 수집
      ↓
4. 예측 vs 실제 비교 → 오차 크면 자동 재학습 트리거
      ↓
5. 새 모델 성능 비교 → 좋으면 자동 배포, 나쁘면 롤백
      ↓
6. 전 과정 실시간 모니터링 대시보드 표시
```

### 5점 핵심 요소

| 요소 | 설명 | LLM 필요? |
|------|------|-----------|
| **전 과정 자동화** | 데이터 수집 → 배포 무인 운영 | ❌ 불필요 |
| **자동 제어** | 예측 결과 → 시스템 조치 연동 | ❌ 불필요 |
| **피드백 루프** | 실제 결과 → 재학습 트리거 | ❌ 불필요 |
| **자동 롤백** | 성능 저하 → 이전 모델 복구 | ❌ 불필요 |
| **실시간 모니터링** | 성능/상태 대시보드 | ❌ 불필요 |

> **참고:** 5점은 LLM 없이 MLOps + 룰 기반 자동화로 달성 가능
