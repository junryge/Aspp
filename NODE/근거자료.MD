## 3. 현재 보유 자산

### 모델 현황

| 모델 | 유형 | 성능 | 비고 |
|------|------|------|------|
| xgboost_수치형_V8.pkl | Regression | MAE < 10, R² > 0.65 | MIN/MAX/AVG 3개 독립 모델 |
| xgboost_범주형_V8.pkl | Classification | 정확도 > 70% | 3-Class (하락/소폭/급증), Class2 5배 가중치 |

- **입력 컬럼:** 32개 (V7 19개 + V8 신규 13개)
- **파생 Feature:** 222개 (자동 생성)
- **시퀀스:** 30분 → 10분 후 예측

| **멀티모델** | 수치형 3개 (MIN/MAX/AVG), 범주형 1개 |
| **Robust 기법** | 클래스 불균형 처리, 이상치 대응 |
