## FABJOB 전문 지식

### 정의
FABJOB = FAB간 이송 작업
여러 설비를 경유: STB → OHT → Conveyor → Lifter → OHT → STB

### 메시지 흐름
VM-FABTRANSPORTJOBCREATED → VM-TRANSPORTJOBLOCATIONCHANGED (여러 번) → VM-TRANSPORTJOBCOMPLETED

### 장비 유형 (장비명 패턴)
- **STB**: 4ANZ*, 6ANZ* (캐리어 보관, Stocker라고 부르지 말것!)
- **OHT**: 4ACM*, 6ACM* (천장 이송)
- **Bridge OHT**: 6ECM* (FAB간 연결 OHT)
- **Conveyor**: 4AFC* (바닥 컨베이어)
- **Lifter**: 6ABL* (층간 이송)

### HCACK 코드 (OHT 응답)
- 0: 성공
- 2: 거부 ⚠️ (OHT 반송 거절 - **주요 지연 원인!**)
- 4: 시작됨
- 6: 실패

### 소요시간 기준
| 구간 | 정상 | 주의 | 경고 |
|------|------|------|------|
| STB → OHT | 3분 | 5분 | 10분 |
| OHT → Conveyor | 3분 | 5분 | 8분 |
| Conveyor 이송 | 8분 | 10분 | 15분 |
| Lifter 층간 | 4분 | 6분 | 8분 |
| OHT → STB | 3분 | 5분 | 8분 |
| **전체** | **20분** | **30분** | **45분** |

### 지연 원인 분석
- **HCACK=2 다발** → OHT 차량 할당 실패 (가장 흔함!)
- OHT 이송 지연 → Rail 혼잡 또는 Rail Cut
- Conveyor 지연 → 적체
- Lifter 지연 → 크레인 점유

### 핵심
**STB → OHT 구간**에서 HCACK=2 거절이 발생하면 큰 지연!
(예: R3 케이스에서 32분 지연)
