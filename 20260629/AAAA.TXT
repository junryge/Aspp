// $\rightarrow$ 같은 게 raw 로 보이던 문제 해결 — HTML 문자열을 받아 기호로 치환해 돌려준다.
function _awDelatex(html){
  var map={'\\rightarrow':'→','\\Rightarrow':'⇒','\\to':'→','\\leftarrow':'←','\\Leftarrow':'⇐',
    '\\geq':'≥','\\ge':'≥','\\leq':'≤','\\le':'≤','\\neq':'≠','\\ne':'≠','\\approx':'≈',
    '\\times':'×','\\cdot':'·','\\pm':'±','\\div':'÷','\\infty':'∞','\\alpha':'α','\\beta':'β',
    '\\sum':'∑','\\rightarrowtail':'→'};
  var h=String(html||'');
  h=h.replace(/\\text\s*\{([^}]*)\}/g,'$1').replace(/\\mathrm\s*\{([^}]*)\}/g,'$1');
  for(var k in map){ h=h.split(k).join(map[k]); }
  // 남은 인라인/블록 수식 구분자 제거 ($$...$$ , $...$)
  h=h.replace(/\$\$([\s\S]+?)\$\$/g,'$1').replace(/\$([^$\n]+?)\$/g,'$1');
  return h;
}

// _trimmedHistory._cut: 실제로 잘린 메시지가 있었는지 기록(저장 마크 판단용)
function _trimmedHistory(history, cap){
  var cut=false;
  var out=(history||[]).map(function(m){
    var c=(m&&typeof m.content==='string')?m.content:'';
    if(c.length>cap){ c=c.slice(0,cap)+'\n\n…(저장공간 절약을 위해 일부 생략됨 — 화면에는 전체가 표시됩니다)'; cut=true; }
    return {role:(m&&m.role)||'assistant', content:c};
  });
  _trimmedHistory._cut=cut;
  return out;
}
// 세션 저장(용량초과 방어): 현재 세션은 반드시 지킨다.
// 1단계: 본문당 상한을 점점 낮춰 저장(평소에도 거대 CSV·리포트 덤프는 캡 → 같은 용량에 세션 더 많이 보관).
// 2단계: 그래도 안 들어가면 '오래된 다른 세션'부터 LRU 삭제 후 재시도.
// 반환: true(원본급 저장) / 'trim'(축소저장) / 'evict'(오래된 세션 정리 후 저장) / false(실패)
function _trySaveSession(aid, sid, history){
  var caps=[60000, 24000, 12000, 4000, 1200];   // 본문 길이 상한(글자). 60K 이하 메시지는 사실상 원본 저장
  for(var i=0;i<caps.length;i++){
    var hist=_trimmedHistory(history, caps[i]);
    var wasCut=_trimmedHistory._cut;
    try{
      localStorage.setItem(_sessKey(aid,sid), JSON.stringify({messages:hist, trimmed:wasCut}));
      return wasCut ? 'trim' : true;            // 실제 잘림 없으면 원본급(true), 있으면 'trim'
    }catch(err){ /* 더 강하게 트림해 재시도 */ }
  }
  // 마지막 수단: 현재 세션을 살리려 '오래된 다른 세션'부터 하나씩 비우고 재시도(LRU).
  var minHist=_trimmedHistory(history, 1200), evicted=0;
  _trySaveSession._evicted=0;
  while(true){
    var idx=_loadSessIdx(aid);
    var others=idx.filter(function(s){return s.sid!==sid;})
                  .sort(function(a,b){return (a.updated||a.created||0)-(b.updated||b.created||0);});
    if(!others.length) break;                    // 더 지울 다른 세션 없음 → 포기
    var victim=others[0];                         // 가장 오래된 세션
    try{ localStorage.removeItem(_sessKey(aid, victim.sid)); }catch(e){}
    _saveSessIdx(aid, idx.filter(function(s){return s.sid!==victim.sid;}));
    evicted++;
    try{
      localStorage.setItem(_sessKey(aid,sid), JSON.stringify({messages:minHist, trimmed:true}));
      _trySaveSession._evicted=evicted;          // 정리한 세션 수 기록(안내용)
      return 'evict';
    }catch(err2){ /* 아직 부족 → 다음으로 오래된 세션도 비움 */ }
  }
  return false;                                  // 그래도 안 되면 실패
}

  var idx=_loadSessIdx(aid).filter(function(s){return (s.count||0)>0 && s.saved!==false;});
  idx.sort(function(x,y){return (y.updated||0)-(x.updated||0);});

  activeSessionId=_newSid(); return [];
}

// 조합 카드 토글 (드롭다운 대체 — 여러 개 연속 클릭 가능)
function renderComboCards(){
  var box=document.getElementById('agComboList');

  var combos=getCombos();

  }

    var col=c.color||'#6366f1';
    var on=activeComboIds.has(c.id);
    var d=document.createElement('div');
    d.className='kcard';
    d.dataset.value=c.id;
    d.title=(c.desc||c.label||c.id);
    d.innerHTML='<span class="kchk">✓</span><span class="kic">🛠</span>'
      +'<span class="knm">'+esc(c.label||c.id)+'</span>'

    function paint(sel){
      d.classList.toggle('sel', sel);
      var chk=d.querySelector('.kchk');
      if(sel){
        d.style.borderColor=col; d.style.background=col+'21'; d.style.boxShadow='0 0 0 1px '+col+'59';

      else activeComboIds.add(c.id);

// 선택한 지식 파일들의 내용을 가져와 시스템 프롬프트용 컨텍스트 블록 생성
var _KCTX_PER_FILE = 4000, _KCTX_TOTAL = 14000;
async function buildKnowledgeContext(files){
  if(!files || !files.length) return '';
  var parts=[], total=0;
  for(var i=0;i<files.length;i++){
    try{
      var r=await fetch('/api/knowledge/view/'+encodeURIComponent(files[i])+'?user_id='+encodeURIComponent(uid()));
      if(!r.ok) continue;
      var d=await r.json();
      var c=String(d.content||'').slice(0,_KCTX_PER_FILE);
      if(total+c.length > _KCTX_TOTAL) c=c.slice(0, Math.max(0,_KCTX_TOTAL-total));
      if(!c) break;
      parts.push('--- 📄 '+files[i]+' ---\n'+c);
      total+=c.length;
      if(total>=_KCTX_TOTAL) break;
    }catch(e){}
  }
  if(!parts.length) return '';
  return '\n\n=== 선택한 내 지식 문서 ===\n'
    +'아래 문서 내용을 우선 근거로 답하세요. 문서에 없는 내용은 추측하지 말고 일반 지식임을 밝히세요.\n\n'
    +parts.join('\n\n');
}

function _dateGroup(ts){
  var d=new Date(ts||0), now=new Date();
  var ymd=function(x){return x.getFullYear()+'-'+(x.getMonth()+1)+'-'+x.getDate();};
  if(ymd(d)===ymd(now)) return '오늘';
  var y=new Date(); y.setDate(now.getDate()-1);
  if(ymd(d)===ymd(y)) return '어제';
  return d.getFullYear()+'.'+String(d.getMonth()+1).padStart(2,'0')+'.'+String(d.getDate()).padStart(2,'0');
}
function _hhmm(ts){ var d=new Date(ts||0); return String(d.getHours()).padStart(2,'0')+':'+String(d.getMinutes()).padStart(2,'0'); }

// 🔮 헤르메스: 수동 스킬 저장 — 모델이 hermes:skill 블록을 안 뱉어도 이 답변을 직접 스킬화
//   답변 버블 밑에 '📌 스킬로 저장' 버튼을 달고, 누르면 인라인 폼(이름/용도/본문)으로 /skill/confirm 호출.
// ── 스킬 이름·설명 자동 추정 (오프라인·무에러, 백엔드 skills.py 와 동일 규칙) ──
function _skillBanned(n){ return /(오늘|today|temp|tmp|test|fix-?\d|수정|어제|에러|error-|bug-\d|\d{4}-\d{2}-\d{2})/i.test(n); }
function _skillNameValid(n){
  n=(n||'').trim().toLowerCase();
  if(!/^[a-z0-9]+(?:-[a-z0-9]+)*$/.test(n)) return false;
  if(n.length<2 || n.length>40) return false;
  if(_skillBanned(n)) return false;
  return true;
}
function _skillSlug(s){
  var stop={the:1,and:1,for:1,with:1,that:1,this:1,you:1,are:1,was:1,csv:1,html:1,http:1,https:1,com:1};
  var toks=(s||'').toLowerCase().match(/[a-z0-9]{2,}/g)||[];
  toks=toks.filter(function(t){ return !stop[t]; });
  return toks.slice(0,4).join('-').slice(0,40).replace(/^-+|-+$/g,'');
}
function _suggestSkillMeta(bodyText){
  var txt=String(bodyText||''), when='';
  var lines=txt.split('\n');
  for(var i=0;i<lines.length;i++){
    var ln=lines[i].replace(/^#+\s*/,'').replace(/[*_`>|]/g,'').trim();
    if(ln.length>=4){ when=ln.slice(0,60); break; }
  }
  var name=_skillSlug((when||txt).slice(0,200));
  if(!_skillNameValid(name)) name='skill-'+String(Date.now()).slice(-6);
  return {name:name, when:when||'재사용 스킬'};
}

    else if(res && res.textContent.indexOf('이름:')===0){ res.textContent=''; }
  }
  input.addEventListener('input', chk); chk();
}

    sysPrompt += '\n\n=== 예시문 참조 (아래는 답변 형식·내용 참고용 예시일 뿐, 스킬·규칙 아님. 실제 요청·데이터를 우선하라) ===\n' + _exRef.trim();
  }
  // 선택 지식문서는 백엔드(RAG/BM25)가 질문 관련 청크만 골라 주입함 → 프론트 주입 제거.
  // (buildKnowledgeContext 는 RAG 미사용 환경 롤백용으로 함수만 보존)
  if(hermesOn){
    bubble.textContent='🔮 헤르메스 기억 불러오는 중…';
    try{
      var _hpr=await fetch('/api/hermes/prep',{method:'POST',headers:{'Content-Type':'application/json'},body:JSON.stringify({user_id:uid(), query:text})});
      var _hpd=await _hpr.json();
      if(_hpd.system_addon) sysPrompt += (sysPrompt?'\n\n':'') + _hpd.system_addon;
    }catch(e){}
  }
  bubble.textContent='';

  // API로 보낼 메시지: 저장본은 그대로 두고, 마지막 user 메시지에만 '한국어/사고과정 차단' 지시만 덧붙임
  // (보고서 스킬 형식은 건드리지 않음 — 영어 사고과정 누출만 막는다)
  var _apiMsgs = chatHistory.map(function(m){ return {role:m.role, content:m.content}; });
  for(var _i=_apiMsgs.length-1; _i>=0; _i--){
    if(_apiMsgs[_i].role==='user'){
      _apiMsgs[_i].content += '\n\n(지시: 한국어로 답하고, 영어 사고과정이나 <think> 태그는 출력하지 마라.)';
      break;
    }
  }

  var payload={
    env: (Array.isArray(activeAgent.envs)&&activeAgent.envs.length)? activeAgent.envs : ['auto'],
    messages: _apiMsgs,
    skills: skills,
    effort: (activeAgent.effort!=null)? activeAgent.effort : 2,
    format: 'auto',
    writing_style: 'auto',
    system_prompt: sysPrompt,
    think_mode: document.getElementById('agThink') ? document.getElementById('agThink').checked : false,
    ctx_guard: document.getElementById('agCtxGuard') ? document.getElementById('agCtxGuard').checked : true,
    knowledge_scripts: (activeAgent.knowledge_scripts || []),
    knowledge_files: (activeAgent.knowledge_files || []),
    user_id: uid(),
    stream: true
  };

  chatAbort=new AbortController();

  try{
    var resp=await fetch('/api/chat',{
      method:'POST', headers:{'Content-Type':'application/json'},
      signal: chatAbort.signal, body: JSON.stringify(payload)
    });
    if(!resp.ok || !resp.body){
      var det=''; try{ det=await resp.text(); }catch(_){}
      bubble.textContent='❌ 요청 실패: HTTP '+resp.status+(det?('\n'+det.slice(0,300)):'');
    } else {
      var reader=resp.body.getReader();
      var dec=new TextDecoder();
      var buf='';
      var errInfo=null;
      outer: while(true){
        var ch; try{ ch=await reader.read(); }catch(_){ break; }
        if(ch.done) break;
        buf+=dec.decode(ch.value,{stream:true});
        var lines=buf.split('\n');
        buf=lines.pop()||'';
        for(var i=0;i<lines.length;i++){
          var s=lines[i].trim();
          if(s.indexOf('data:')!==0) continue;
          var pl=s.slice(5).trim();
          if(!pl) continue;
          if(pl==='[DONE]') break outer;
          try{
            var obj=JSON.parse(pl);
            if(obj.type==='token' && obj.t){
              acc+=obj.t;
              // 추론(</think> 이전)은 가리고, 아직 추론 중이면 안내만 표시
              if(acc.indexOf('</think>')>=0){ bubble.textContent=stripThink(acc); }
              else if(/<think>/.test(acc)){ bubble.textContent='💭 생각하는 중…'; }
              else { bubble.textContent=acc; }
              var c=document.getElementById('chatMsgs'); c.scrollTop=c.scrollHeight;
            } else if(obj.type==='error'){ errInfo=obj; }
              else if(obj.type==='end'){ if(obj.truncated) truncatedFlag=true; }
          }catch(_){}
        }
      }
      var errLine = errInfo ? ('\n\n❌ '+(errInfo.error||'')+(errInfo.detail?('\n'+errInfo.detail):'')) : '';
      var disp = stripThink(acc);
      var pending = null;
      if(hermesOn && acc.trim()){
        try{
          var _por=await fetch('/api/hermes/post',{method:'POST',headers:{'Content-Type':'application/json'},
            body:JSON.stringify({user_id:uid(), answer:acc, user_message:text, session_id:(activeAgent.agent_id||'')})});
          var _pod=await _por.json();
          if(typeof _pod.clean==='string') disp=_pod.clean;
          if(_pod.questions && _pod.questions.length) disp += (disp?'\n\n':'')+'❓ '+_pod.questions.join('\n❓ ');
          if(_pod.pending_skills && _pod.pending_skills.length) pending=_pod.pending_skills;
        }catch(e){}
      }
      bubble.style.whiteSpace='normal';
      bubble.innerHTML = renderFinal(disp + errLine);
      _awAppendMdBar(bubble, disp + errLine, skills);  // md 스킬 + 펜스 없으면 다운로드 바 폴백
      chatHistory.push({role:'assistant', content: disp});
      saveChat();
      if(pending) _awSkillCards(pending);
      _awContinueBar(bubble, truncatedFlag);  // ⏭ 이어서 보기 버튼
      _awManualSkillBtn(bubble, disp);         // 📌 이 답변을 직접 스킬로 저장
    }
  }catch(e){
    if(e && e.name==='AbortError'){
      var _p=stripThink(acc);
      bubble.innerHTML = renderFinal((_p?_p:'') + '\n\n⏹ 중단됨');
      if(_p){ chatHistory.push({role:'assistant', content:_p}); saveChat(); }
    }
    else { bubble.textContent=acc+'\n\n❌ '+(e.message||e); }
  } finally {
    isStreaming=false; sendBtn.disabled=false; sendBtn.textContent='전송';
    sendBtn.style.background=''; chatAbort=null;
  }
}
