# AMHS 장애 예측모델 개발 구체화

---

## 1. 사용 Machine Learning 모델

| 구분 | 알고리즘 | 목적 |
|------|----------|------|
| **수치형** | XGBoost Regressor | 10분 후 변화량 숫자 예측 (MIN/MAX/AVG) |
| **범주형** | XGBoost Classifier | 10분 후 변화 클래스 분류 (3-Class) |

**선정 이유:** 정형 데이터 예측 성능 우수, GPU 가속 지원, Feature 중요도 해석 가능

---

## 2. 검증 현황

### 2.1 검증 방법

| 항목 | 적용 여부 |
|------|----------|
| Train/Test 분할 (80/20) | ✅ 적용 |
| 별도 테스트셋 평가 | ✅ 적용 |
| K-Fold Cross Validation | ❌ 미적용 |

### 2.2 검증 성능 결과

| 모델 | 지표 | 목표 | 달성 |
|------|------|------|------|
| 수치형 | MAE (평균 오차) | < 10 | ✅ |
| 수치형 | R² (결정계수) | > 0.65 | ✅ |
| 범주형 | 정확도 | > 70% | ✅ |
| 범주형 | 급증(Class2) Recall | > 65% | ✅ |

## 3. 모델 상세

| 항목 | 내용 |
|------|------|
| **입력** | 32개 컬럼 → 222개 Feature 자동 생성 |
| **시퀀스** | 과거 30분 데이터 |
| **예측** | 10분 후 (15분/25분 확장 가능) |
| **특수 기법** | 클래스 불균형 가중치 적용 (급증 5배) |
