🧮 AI 모델의 수학적 구조

1. 데이터 입력 구조

입력 데이터 수식
입력: X = [물류량(t-29), 물류량(t-28), ..., 물류량(t-1), 물류량(t)]
출력: Y = 물류량(t+10분)

여기서 t = 현재 시간

🚗 고속도로 비유

지난 30분간 톨게이트를 통과한 차량 수를 보고, 10분 후 통과할 차량 수를 예측하는 것과 같습니다.

2. LSTM 신경망 구조

4층 LSTM 네트워크 구조

입력층
30개 시점 × 1개 특징 (물류량)
↓
LSTM 층 1 (100 유닛)
수식: h₁ = σ(W₁·X + b₁) × tanh(C₁)
Dropout: 20% (과적합 방지)
↓
LSTM 층 2 (100 유닛)
수식: h₂ = σ(W₂·h₁ + b₂) × tanh(C₂)
Dropout: 20%
↓
LSTM 층 3 (100 유닛)
수식: h₃ = σ(W₃·h₂ + b₃) × tanh(C₃)
Dropout: 20%
↓
LSTM 층 4 (100 유닛)
수식: h₄ = σ(W₄·h₃ + b₄) × tanh(C₄)
Dropout: 20%
↓
출력층 (Dense)
수식: Y = W₅·h₄ + b₅
1개 값 출력 (10분 후 예측값)

3. LSTM 게이트 메커니즘

LSTM 핵심 수식
잊기 게이트: f_t = σ(W_f · [h_{t-1}, x_t] + b_f)
입력 게이트: i_t = σ(W_i · [h_{t-1}, x_t] + b_i)
출력 게이트: o_t = σ(W_o · [h_{t-1}, x_t] + b_o)
셀 상태:    C_t = f_t × C_{t-1} + i_t × C̃_t
히든 상태:   h_t = o_t × tanh(C_t)

여기서 σ = 시그모이드 함수, W = 가중치, b = 편향

👨‍💼 기억력 있는 관리자 비유

잊기 게이트: "2시간 전 데이터는 덜 중요하니 20% 정도만 기억해"
입력 게이트: "방금 들어온 신규 주문이 중요해"
출력 게이트: "종합해서 다음 부서에 이 정보를 전달해"

4. 양방향 LSTM (Bi-LSTM) 구조

Bi-LSTM 수식
순방향: →h_t = LSTM_forward(x_t, →h_{t-1})
역방향: ←h_t = LSTM_backward(x_t, ←h_{t+1})
결합:   h_t = [→h_t ; ←h_t]

• 양방향으로 시계열 분석
• 과거와 미래 정보 동시 활용
• 2층 × 50유닛 × 2방향 = 100차원 출력

🔄 양방향 교통 분석 비유

고속도로의 상행선과 하행선을 동시에 분석하여 전체 교통 흐름을 파악하는 것처럼, Bi-LSTM은 시간의 순방향과 역방향을 모두 분석하여 더 정확한 패턴을 포착합니다.

📊 데이터 전처리 과정

표준화 (Standardization)

StandardScaler 수식
정규화: X_scaled = (X - μ) / σ

여기서:
- μ (평균) = 1714개
- σ (표준편차) = 287개
- 원본 범위: 800~2500개
- 정규화 범위: 약 -3 ~ +3

🌡️ 온도계 비유

섭씨 온도(800~2500)를 표준 점수(-3~+3)로 변환하는 것과 같습니다. AI가 다양한 범위의 숫자를 동일한 기준으로 이해할 수 있게 됩니다.

🔄 실시간 예측 프로세스

전체 예측 수식
1. 데이터 수집: X = [물류량(t-29), ..., 물류량(t)]
                ↓
2. 정규화: X_norm = (X - 1714) / 287
                ↓
3. LSTM 처리: h = LSTM₄(LSTM₃(LSTM₂(LSTM₁(X_norm))))
                ↓
4. 예측 생성: Y_norm = Dense(h)
                ↓
5. 역정규화: Y = Y_norm × 287 + 1714
                ↓
6. 최종 예측: "10분 후 예상 물류량 = Y개"
