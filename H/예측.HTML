🔄 주요 개선사항

1. 하이브리드 앙상블 모델

단일 LSTM에서 5개 모델의 앙상블로 확장하여 예측 정확도와 안정성을 크게 향상시켰습니다.

  • LSTM: 30% 가중치
  • GRU: 25% 가중치
  • RNN: 15% 가중치
  • Bi-LSTM: 30% 가중치
  • ARIMA: 보조 예측

2. CPU 최적화

명시적 CPU 모드 설정으로 GPU가 없는 환경에서도 안정적으로 실행 가능합니다.

os.environ['CUDA_VISIBLE_DEVICES'] = '-1' tf.config.set_visible_devices([], 'GPU')

3. 실시간 예측 기능

실시간 데이터 스트림에 대한 즉각적인 예측이 가능한 predict_realtime() 함수를 추가했습니다.

  • 마지막 30분 데이터만으로 예측
  • 10분 후 물류량 예측
  • 병목 발생 사전 경고

4. 고급 병목 분석

단순 임계값 비교에서 벗어나 예측 기반의 사전 병목 감지 시스템을 구현했습니다.

  • 상위 15% 기준 동적 임계값
  • 10분 전 사전 경고
  • 병목 발생 확률 제공

5. 종합적 성능 평가

다양한 성능 지표를 통해 모델의 예측 품질을 종합적으로 평가합니다.

  • MAE, MSE, RMSE, R²
  • 개별 모델별 성능 비교
  • 시각화된 성능 리포트

6. 향상된 로깅 및 모니터링

상세한 로깅과 진행 상황 모니터링으로 디버깅과 운영이 용이해졌습니다.

  • 구조화된 로그 메시지
  • 파일 및 콘솔 동시 출력
  • 에러 추적 및 복구 지원
