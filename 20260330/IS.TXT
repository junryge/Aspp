# Blender LLM 3D Assistant 사용법
## SK Hynix 내부 LLM 연동 Blender 자연어 제어 애드온

---

## 1. 준비물

- **Blender 3.6+** (회사 PC에 설치)
- **token.txt** (내부 LLM API 인증 토큰 파일)
- **blender_hynix_llm.py** (이 애드온 파일)

---

## 2. 설치 (1분)

### Step 1: token.txt 준비
기존에 쓰던 token.txt를 알기 쉬운 경로에 복사:
```
C:\token.txt
```
또는 원하는 경로 아무 데나 OK.

### Step 2: Blender에 스크립트 로드
1. Blender 실행
2. 상단 메뉴 → **Scripting** 워크스페이스 탭 클릭
3. **열기(Open)** → `blender_hynix_llm.py` 선택
4. **Alt + P** (또는 ▶ 실행 버튼) 눌러서 스크립트 실행

### Step 3: 패널 확인
1. **3D Viewport**로 돌아가기 (Layout 탭)
2. **N키** 눌러서 사이드바 열기
3. **"LLM Assistant"** 탭 확인

---

## 3. 설정

패널 상단 "설정" 영역:

| 항목 | 기본값 | 설명 |
|------|--------|------|
| API Base | `http://dev.hcp.llm.skhynix.com` | 변경 불필요 |
| Token 경로 | `C:/token.txt` | 본인 token.txt 경로 |
| 모델 | `Qwen3-Coder-Next` | 코딩 특화 모델 |

**"모델 목록 새로고침"** 버튼 → 서버 모델 조회 + 자동으로 Coder 모델 선택됨.
(전체 목록은 Blender 콘솔창에서 확인: 상단 Window → Toggle System Console)

---

## 4. 사용법

### 기본 흐름
1. **"자연어 명령"** 칸에 한국어로 입력
2. **"실행"** 클릭
3. 기다리면 3D 뷰포트에 결과 나옴

### 잘 되는 명령 예시 (구체적으로!)

```
빨간 큐브 만들어줘
```
```
파란 구 5개를 x축으로 간격 3으로 배치해줘
```
```
선택된 오브젝트 색을 노란색으로 바꿔줘
```
```
모든 오브젝트 삭제해줘
```
```
원형으로 실린더 12개 배치해줘 반지름 10
```
```
선택된 오브젝트에 메탈 재질 적용해줘 roughness 0.2
```
```
큐브로 계단 모양 10단 만들어줘
```
```
격자 형태로 큐브 100개 만들어줘 랜덤 색상 랜덤 크기
```

### 안 되는 명령 (너무 추상적)

```
❌ 예쁜 집 만들어줘
❌ 캐릭터 모델링 해줘
❌ 멋있게 꾸며줘
```

### 핵심 팁: 구체적으로 말하기

| 나쁜 예 | 좋은 예 |
|---------|---------|
| 집 만들어줘 | 큐브로 벽 4개, 위에 피라미드 지붕 올려줘 |
| 도시 만들어줘 | 랜덤 높이 큐브 20개를 격자로 배치해줘 |
| 예쁘게 해줘 | 오브젝트 색을 파란색으로, roughness 0.1로 |

---

## 5. 에러 발생 시

- **"API 실패: 연결 실패"** → token.txt 경로 확인, 사내망 연결 확인
- **"코드 추출 실패"** → 프롬프트를 더 구체적으로 다시 시도
- **"문법 에러"** → 자동으로 3회 재시도함. 그래도 안 되면 다르게 표현
- **콘솔 확인**: Window → Toggle System Console에서 상세 로그 확인 가능

---

## 6. 구조

```
너 (한국어 자연어)
    ↓
Blender 애드온 (이 스크립트)
    ↓ HTTP POST /v1/chat/completions
내부 LLM (Qwen3-Coder-Next)
    ↓ bpy Python 코드 응답
코드 추출 + 안전성 검사 + 문법 검증
    ↓
exec() 실행 → Blender 3D 뷰포트에 반영
    ↓ (에러 시)
에러 메시지로 LLM에 수정 요청 (최대 3회)
```
