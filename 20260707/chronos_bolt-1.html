Chronos-Bolt 란?

약 1,000억 개 시계열로 사전학습해서, 처음 보는 시계열에도 학습 없이 예측을 뽑는 파운데이션 모델. 원본 Chronos를 빠르고 가볍게 다시 설계한 2세대.

250× faster than original Chronos
Chronos (original)Chronos-Bolt

01 정체 — 한 문단

NLP 언어모델이 대량 텍스트로 학습해 처음 보는 문장을 처리하듯, Chronos-Bolt는 약 1,000억 개 시계열 관측치로 학습해 학습 때 본 적 없는 시계열도 예측한다. 이게 zero-shot 예측이다. 구조는 구글 T5 encoder-decoder 기반이고, 원본 Chronos를 속도·메모리 측면에서 다시 짠 개선판이다.

02 계보 — Chronos 패밀리

세대이름핵심 방식
1세대Chronos (원본)시계열을 토큰화 → LLM처럼 미래를 샘플링 (autoregressive). 느리고 무거움
2세대Chronos-Bolt시계열을 패치로 묶어 인코딩 → 여러 스텝 quantile을 한 번에 생성
3세대Chronos-2후속 최신판 (amazon/chronos-2)

원본 논문: Chronos: Learning the Language of Time Series (Ansari et al., 2024, TMLR).

03 원본 vs Bolt — 뭐가 바뀌었나

구분원본 ChronosChronos-Bolt
입력 처리관측치를 개별 토큰으로 양자화패치로 묶어 입력
생성 방식한 스텝씩 autoregressive 샘플링direct multi-step (동시 생성)
출력샘플 경로에서 분포 추정quantile 직접 출력
속도기준최대 250배

Chronos-Bolt (Base) 가 원본 Chronos (Large) 보다 정확하면서 600배 이상 빠르다 — 작은 모델이 큰 원본을 이긴다.

04 아키텍처 · 동작 원리

1 · Input과거 시계열(context)을 여러 관측치 묶음인 패치로 분할
2 · T5 Encoder패치를 인코더에 넣어 시계열 표현(representation) 학습
3 · T5 Decoder · Direct Multi-step미래 여러 스텝의 quantile 예측을 한 번에 생성 (스텝별 반복 없음 → 빠름)
4 · Output확률적 예측: 점 추정 + 분위수(0.1 / 0.5 / 0.9 등). shape [num_series, num_quantiles, prediction_length]

07 사용법

(A) AutoGluon — 프로덕션 권장 (파인튜닝·covariate·앙상블 지원) 
# pip install autogluon
from autogluon.timeseries import TimeSeriesPredictor, TimeSeriesDataFrame

df = TimeSeriesDataFrame(".../m4_hourly/train.csv")

predictor = TimeSeriesPredictor(prediction_length=48).fit(
    df,
    hyperparameters={"Chronos": {"model_path": "amazon/chronos-bolt-base"}},
)
predictions = predictor.predict(df)
(B) chronos-forecasting — 연구/최소 인터페이스 
# pip install chronos-forecasting
import torch, pandas as pd
from chronos import BaseChronosPipeline

pipeline = BaseChronosPipeline.from_pretrained(
    "amazon/chronos-bolt-base",
    device_map="cuda",        # "cpu" 또는 "mps"
    torch_dtype=torch.bfloat16,
)
# 출력 shape: [num_series, num_quantiles, prediction_length]
forecast = pipeline.predict(context=torch.tensor(series), prediction_length=12)
